# Selection

The Telerik UI MultiViewCalendar allows the user to select multiple dates by using the multiple-view MultiViewCalendar selection mode.

```Razor
      @(Html.Kendo().MultiViewCalendar()
          .Name("MultiViewCalendar")
          .Selectable("range")
      )
```
